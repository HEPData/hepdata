Upload an archive (.zip, .tar, .tar.gz, .tgz) containing YAML files formatted according to these guidelines. An example submission archive is available here. You can validate your YAML files offline using this script.

We also accept a single YAML file (.yaml or .yaml.gz) containing all of the submission data. For records uploaded to the old HepData site, this format can be obtained automatically by appending "/yaml" to the old record URL.

Alternatively, upload a single text file with extension .oldhepdata containing the "input" format that was used for data submissions from the old HepData site (see sample).
